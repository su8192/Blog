### 📋 软件概述

> 📹土豆录屏是一款 windows 平台下的屏幕录制软件，由国内的土豆软件工作室开发，支持多种录屏方式，比如电脑屏幕录制、游戏画面录制、摄像头录制等，是一款简单易用、功能强大的录屏软件。

### ✨ 核心特性

软件界面简洁，操作简单：在主界面上预览画面，直接开始录制； 

录屏模式多样化：支持全屏录制、游戏录制、摄像头+声音录制，自定义窗口大小录制也不在话下； 

视频处理方便：录制完成后可以直接对视频进行处理，比如截取其中一个或多个片段； 

支持计划录制：可以设置好录屏时间，到了该时间点会自动录制并保存； 

支持 4K 和最高 120 FPS 的画质输出，录屏质量随需更改，多种视频、音频码率调整，水印添加等； 

不需要登录账号，打开软件直接使用，没有任何广告和使用限制。 


### 💽 快速安装

📍运行安装程序直接点击安装即可。

需要注意的是，土豆录屏运行时依赖 Microsoft Visual C++ 组件

如果电脑里没有的话，在安装软件时会提示安装这个  Microsoft Visual C++ 组件
